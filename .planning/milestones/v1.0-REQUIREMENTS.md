# Requirements Archive — v1.0 Type Matchup Milestone

**Archived:** 2026-04-13  
**Status:** ✅ SHIPPED (29/31 satisfied, 2 partial — see audit)

## Overview

Made Pokémon types mechanically meaningful in gym and Elite Four battles. The player's team composition influences wheel odds — type advantage creates stakes, strategy, and narrative drama that pure luck cannot.

## Out of Scope — Confirmed Exclusions

| Item | Disposition |
|------|-------------|
| Champion battle type matchups | Excluded by user — v2+ |
| Rival battle type matchups | Excluded by user — v2+ |
| Move-level type calculation | By design — team-level only |
| Net advantage (attack vs defence) | By design — any SE attack = "strong" |
| `HttpClient` fetch for type-matchups.json | Static import chosen — zero-latency |
| TypeIconPipe extraction to shared utility | Deferred — standalone refactor |
