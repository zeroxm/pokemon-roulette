## Milestone Summary

Delivered 7 corrections to the v1.0 Pokédex feature: fixed button layout, enforced grid column counts, closed 2 missing event hooks (evolution + trade), created complete generation data covering all 9 gens including legendaries/mythicals/Ultra Beasts/Paradox Pokémon, and renamed the Local Dex tab to a dynamic "{Region} Dex" label with full i18n support across 6 locales.

## Key Decisions

- Single insertion point in `replaceForEvolution()` covers all evolution paths (evolvePokemon, evolveSecondPokemon ×2, inline) rather than per-callsite hooks
- `Array.from` for consecutive national dex ranges — concise, no inlined number arrays
- ngx-translate `{{region}}` interpolation for tab label — proper nouns (region names) need no translation, word order varies by locale (EN: `{{region}} Dex`, DE: `{{region}}-Pokédex`, others: `Pokédex {{region}}`)
- `pokedex-by-generation.ts` placed in `src/app/pokedex/` to colocate with dex components, not in roulette directory
