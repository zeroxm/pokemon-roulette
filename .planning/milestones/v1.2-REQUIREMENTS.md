# Requirements Archive — v1.2 Pokédex Visual Enhancement

**Status:** ✅ ALL DELIVERED — 2026-04-08
**Version:** v1.2
**Milestone:** Pokédex Visual Enhancement

## Requirements

### Bug Fixes

- [x] **MOB-01**: The Pokédex modal uses `windowClass: 'modal-fullscreen-sm-down'` but this class is not taking effect on mobile devices. **Delivered:** Global `@media (max-width: 576px)` override in `src/styles.css` — ng-bootstrap portal rendering requires global CSS. Phase 6.

### Pokémon Detail Modal

- [x] **DETAIL-01**: When a user taps/clicks a **seen or won** Pokémon cell in the Pokédex grid (unseen/unknown cells are NOT clickable), a detail modal opens showing: the Pokémon's official-artwork sprite inside a circle element, the Pokédex number (zero-padded, e.g. `#025`), and the Pokémon's localized name via ngx-translate. **Delivered:** `PokedexEntryClickEvent` + `PokedexDetailModalComponent` + `(entryClicked)` wiring. Phase 7.

- [x] **DETAIL-02**: The detail modal shows alternate form sprites when available. Use the existing captures/form implementation in the project as reference. **Delivered:** `PokemonFormsService.getFormIds(id).length > 1` check; form buttons rendered via `@for`. Phase 7.

### Shiny Tracking

- [x] **SHINY-01**: Extend `PokedexData` caught entry schema to include `shiny: boolean`. When `markSeen(pokemonId, shiny)` is called with `shiny: true`, persist it. Once set, never reverts. **Delivered:** `PokedexEntry.shiny?: boolean`; upgraded `markSeen` with accumulation logic; `markWon` spread fix preserves flag; all 5 call sites pass `.shiny`. Phase 6.

- [x] **SHINY-02**: In the Pokémon detail modal, if the Pokédex entry has `shiny: true`, display a "Shiny" toggle button. Toggling switches the official-artwork image between regular and shiny. **Delivered:** `@if (hasShinyToggle)` in modal template; `showShiny` toggle; `artworkUrl` getter switches URL. Phase 7.

### Visual Polish

- [x] **VIS-01**: Replace static gold border/box-shadow on "won" Pokémon cells with a CSS keyframes glow-pulse animation. **Delivered:** `@keyframes glow-pulse` with dim/bright gold glow cycle, 2s ease-in-out infinite. Phase 6.
