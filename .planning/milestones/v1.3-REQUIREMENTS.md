# Requirements: Pokémon Roulette Pokédex — v1.3

**Defined:** 2026-04-09
**Core Value:** Players can see which Pokémon they've encountered and won with over time, giving the game lasting replayability beyond individual runs.

## v1.3 Requirements

### Shiny Persistence

- [x] **SHINY-03**: Pokédex entry for a newly caught Pokémon reflects `shiny: true` after the shiny roulette resolves with a shiny result

### Alt-Form Champion Wins

- [x] **ALTW-01**: Beating the Champion with an alt-form Pokémon on the team marks both the alt-form ID and the base national dex ID as `won: true` in the Pokédex
