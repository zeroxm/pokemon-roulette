|----------|-------------|
| PokeAPI CDN → img[src] | Type icon URLs constructed from enum values, not user input |
| component state → template | matchupAdvantageTypes/matchupDisadvantageTypes are PokemonType[] — no raw user strings rendered as HTML |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-03 | Spoofing | img[src] binding for type icons | accept | URL constructed from `pokemonTypeDataByKey[type].id` (enum-constrained integer) — no user-controlled input in URL path |
| T-08-04 | Information Disclosure | Rendered type icons reveal leader type | accept | Leader types are intended game information; no PII involved |
| T-08-05 | Tampering | Injected content via type name in [alt] | accept | `type` is a `PokemonType` enum value — always a known lowercase string like "fire", never user-supplied |
</threat_model>
