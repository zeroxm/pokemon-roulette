# Plan 02: Fix WheelComponent DOM Access (DOM-01)

**Phase:** 7 — Component Hygiene
**Plan:** 02
**Requirement(s):** DOM-01
**Status:** Pending

## Goal
Replace the two `document.getElementById()` calls in `WheelComponent.ngAfterViewInit()` with Angular `@ViewChild` template references, eliminating the direct DOM dependency and making the component testable without a real DOM.

## Files Changed
- `src/app/wheel/wheel.component.html` — add `#wheel` and `#pointer` template reference variables to the two `<canvas>` elements
- `src/app/wheel/wheel.component.ts` — add `ViewChild` + `ElementRef` imports, declare two `@ViewChild` properties, update `ngAfterViewInit()` to use them

## Changes

### `src/app/wheel/wheel.component.html`

Add template reference variables to the existing canvas elements.

**Line 4 — wheel canvas:**
```html
<!-- BEFORE -->
<canvas id="wheel"
        [width]="wheelWidth"
        [height]="canvasHeight"
        (click)="spinWheel()">
</canvas>

<!-- AFTER -->
<canvas id="wheel"
        #wheel
        [width]="wheelWidth"
        [height]="canvasHeight"
        (click)="spinWheel()">
</canvas>
```

**Line 9 — pointer canvas:**
```html
<!-- BEFORE -->
<canvas id="pointer"
        [width]="cursorWidth"
        [height]="canvasHeight">
</canvas>

<!-- AFTER -->
<canvas id="pointer"
        #pointer
        [width]="cursorWidth"
        [height]="canvasHeight">
</canvas>
```

> The `id` attributes are intentionally kept — they may be used for CSS or external references. Only the template refs are added.

---

### `src/app/wheel/wheel.component.ts`

**Line 1 — update Angular core import** to add `ElementRef` and `ViewChild`:
```typescript
// BEFORE:
import { AfterViewInit, Component, EventEmitter, HostListener, Input, OnChanges, Output, SimpleChanges } from '@angular/core';

// AFTER:
import { AfterViewInit, Component, ElementRef, EventEmitter, HostListener, Input, OnChanges, Output, SimpleChanges, ViewChild } from '@angular/core';
```

**After line 28 (after `spinning = false;`) — add the two `@ViewChild` declarations:**
```typescript
@ViewChild('wheel') wheelCanvasRef!: ElementRef<HTMLCanvasElement>;
@ViewChild('pointer') pointerCanvasRef!: ElementRef<HTMLCanvasElement>;
```

**Lines 65–69 — replace `ngAfterViewInit()` body:**
```typescript
// BEFORE:
ngAfterViewInit(): void {
  this.wheelCanvas = <HTMLCanvasElement>document.getElementById('wheel');
  this.wheelCtx = this.wheelCanvas.getContext('2d')!;
  this.pointerCanvas = <HTMLCanvasElement>document.getElementById('pointer');
  this.pointerCtx = this.pointerCanvas.getContext('2d')!;

// AFTER:
ngAfterViewInit(): void {
  this.wheelCanvas = this.wheelCanvasRef.nativeElement;
  this.wheelCtx = this.wheelCanvas.getContext('2d')!;
  this.pointerCanvas = this.pointerCanvasRef.nativeElement;
  this.pointerCtx = this.pointerCanvas.getContext('2d')!;
```

> The `<HTMLCanvasElement>` cast is removed — `ElementRef<HTMLCanvasElement>.nativeElement` is already typed as `HTMLCanvasElement`.

The four lines that follow (`translateService.get(...)` block) are **unchanged**.
